05 / technology

Built on real standards.
Not mock demos.

ERC-8004 Agent Identity
Each of the four agents has a unique on-chain ID, reputation score, and run counter. AI identity standard for autonomous agents.
EIP-712 Typed Consent
HybridVault uses typed data signing so users authorize agent actions — without handing over private keys.
AgentExecutor on-chain
Every AI allocation decision is logged on Mantle via executeAllocation() — auditable, immutable, verifiable.
Pyth Network Price Feeds
YieldOracle uses Pyth Hermes for real-time MNT/USD and RWA asset prices. No mock data — live feeds.
Mantle L2 + IPFS
All contracts on Mantle Sepolia. Asset documents pinned to IPFS via Pinata. Low-cost, high-throughput.
Claude + Groq inference
Claude Sonnet for reasoning, Groq (Llama 4) as fallback. Streaming responses, tool use, multi-agent delegation.
